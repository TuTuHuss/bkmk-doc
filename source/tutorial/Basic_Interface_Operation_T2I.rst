产品使用方法
########################################


1.文生图操作界面简介
****************************************
.. image:: img/base_1.png
   :align: center


** 导航区
----------------------------------------
可以前往网站其他区域

#. 文生图：通过描述词汇进行生成图片

#. 图生图：上传预设图片与描述词汇进行生成图片

#. 局部修改：通过选取部分区域与描述词汇进行生成图片


** 个人资料区
----------------------------------------
#. 当前剩余**灵感点**数量
#. 语言切换
#. 我的资料：在查看历史生成记录
#. 登出

1.1 预览区
----------------------------------------
#. 这边将显示你生成的图片
#. 在首次打开页面时，这边将显示一些我们预设的图片

1.2 描述区
----------------------------------------
#. 这边可编辑与输入您想要描述词（描述词汇教程可前往）
